How does it look?

Cyanide is a theme that keeps the basic look of Discord, but also does a unique spin on it with a more cyan-orientaded color palette. Cyanide works in dark mode, light mode, compact mode and cozy mode!


How do I install it?

Powercord

You can install it for Powercord by copying a command and pasting it in your terminal of your choice. (Windows users: do NOT open it as an administrator!) This command doesn't work directly if you changed the name of your Powercord folder, so you need to replace "powercord" in cd powercord with the name of your folder.

Is there a Discord that I can join?

Yes! Cyanide has its own Discord server, that is optimized for Cyanide, with a custom bot, channel names and even a few special icon!
